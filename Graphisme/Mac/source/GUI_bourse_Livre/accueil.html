Bienvenue dans l'application GBL.

Il semblerait qu'aucun serveur ne soit configuré.
Veuillez suivre les instructions suivantes pour configurer un serveur :
  1. : Cliquez sur Réglages, nous allons configuré le serveur.

  2. Entrez l'adresse IP (ou l'adresse) du serveur web. Cliquez sur entrez pour valider.

  3. : Cliquez sur Home pour charger l'application.

  4. L'application est à présent fonctionelle.

Possibilitées en cas de non-fonctionnement de l'application :
  • Redémarrez l'application.
  • Redémarrez le serveur web.
  • Contactez l'administrateur du réseau.
